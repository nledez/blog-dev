--- layout: post title: Plugin pour Dashboard tags: - Informatique - OpenSource - Web status: publish type: post published: true meta: {} --- Déjà qu'est que Dashboard : http://www.apple.com/macosx/features/dashboard/ En gros c'est un système de petits outils pour faire plein de petits trucs inutiles donc indispensables. J’ai développé un plugin en 1h : http://nicolas.ledez.free.fr/dev/widget/FreeIpv6pt/ Le plugin fait la même chose que la page HTML d'origine. Elle affichait un compteur qui se rafraîchissait automatiquement : http://ipv6pourtous.free.fr/signataires/compteur/ La différence, c'est que le navigateur n'est pas obligatoirement démarré.
