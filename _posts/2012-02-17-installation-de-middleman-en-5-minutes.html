  • Créer un environnement RVM
  • Trouver la version de Middleman à installer
  • Cloner Middleman
  • Installer les bonnes dépendances
  • Installer la bonne version de Middleman
Pourquoi tout ça ? Quand j'ai voulu l'installer, Middleman m'a demandé d'installer plein de dépendances et Gem n'a pas su le faire tout seul avec les bonnes versions donc :

Créer un environnement RVM

$ rvm use 1.9.3@middleman --create

Trouver la version de Middleman à installer

$ gem search --remote middleman Qui va donner : *** REMOTE GEMS *** beans-middleman (1.0.14) guard-middleman (0.1.0) huge-boilerplate-middleman (0.3) middleman (2.0.14 ruby x86-mingw32) middleman-blog (0.1.5) [...] Noter la version 2.0.14 dans notre cas.

Cloner Middleman

$ git clone https://github.com/middleman/middleman.git $ cd middleman $ git checkout v2.0.14

Installer les bonnes dépendances

$ bundle Si Bundler n'est pas installé : $ gem install --no-ri --no-rdoc bundler

Installer la bonne version de Middleman

$ gem install middleman --version 2.0.14 Et voilà ! Votre version de Middleman toute prête à être utilisée.
